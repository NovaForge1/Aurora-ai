# Aurora AI

Aurora is a futuristic AI platform that can generate videos, images, and natural conversations — built to inspire creativity and innovation.

## Features
- AI video and image generation
- Conversational AI interface
- Modern, futuristic design

## How to Run
1. Clone the repo: `git clone https://github.com/yourname/aurora-ai`
2. Open `index.html` in your browser, or run the development server.
3. Enjoy creating with Aurora!
